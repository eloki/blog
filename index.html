Sublime Text3 安装配置

系统环境
Windows 10 专业版

下载安装

http://www.sublimetext.com/3

安装Package Control

使用Ctrl+`快捷键或者通过View->Show Console菜单打开命令行，粘贴代码：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

如果无法使用代码安装，可手动安装Package Control：
1.点击Preferences -> Browse Packages
2.进入打开的目录的上层目录，然后再进入Installed Packages目录
3.下载Package Control.sublime-package并复制到Installed Packages目录
4.重启Sublime Text。

升级

配置中加入”update_check”: false可以禁用自动更新

汉化

点击Preferneces -> Browse Packages ，将会弹出当前的配置文件夹，将3083汉化包内的三个文件夹放到该文件夹内，汉化立刻生效。

插件安装

点击快捷键 Ctrl + Shift + P 输入 Install 点击回车, 输入 插件名称 再点击回车

GBK/GB2312中文编码支持

在Package Control输入ConvertToUTF8,安装一下插件就可以识别了,方法和上面一样

右键菜单

以下内容在安装目录保存为.inf文件，右键->安装

[Version] 
Signature="$Windows NT$" 
[DefaultInstall] 
AddReg=SublimeText3 
[SublimeText3] 
hkcr,"*\\shell\\SublimeText3",,,"用 SublimeText3 打开" 
hkcr,"*\\shell\\SublimeText3\\command",,,"""%1%\sublime_text.exe"" ""%%1"" %%*" 
hkcr,"Directory\shell\SublimeText3",,,"用 SublimeText3 打开" 
hkcr,"*\\shell\\SublimeText3","Icon",0x20000,"%1%\sublime_text.exe, 0" 
hkcr,"Directory\shell\SublimeText3\command",,,"""%1%\sublime_text.exe"" ""%%1"""

MongoDB for Windows安装配置

系统环境
Windows 10 专业版

下载安装

https://www.mongodb.org/downloads

环境变量

C:\Program Files\MongoDB\Server\3.2\bin;

配置

创建数据库和日志存放目录
新建“MongoDB\db”和“MongDB\log”两个文件夹，分别存放数据库文件和日志文件

创建config文件
“C:\Program Files\MongoDB 2.6 Standard\bin”,新建mongo.config文件,内容:

##数据库
dbpath=D:\ProgramData\MongoDB\db
##日志
logpath=D:\ProgramData\MongoDB\log\mongodb.log

添加环境变量

PATH="C:\Program Files\MongoDB\Server\3.2\bin;"

安装服务
以管理员方式打开CMD窗口，安装MongoDB服务

mongod --config "C:\Program Files\MongoDB\Server\3.2\bin\mongo.config" --install

启动服务

net start mongodb

测试连接

mongo

文件结构

C:\Program Files\MongoDB\Server\3.2\bin\
mongo.exe：客户端，支持js语法
mongod.exe：服务端
mongodump.exe：备份工具
mongorestore.exe：恢复工具
mongoexport.exe：导出工具
mongoimport.exe：导入工具
mongostat.exe：实时性能监控工具
mongotop.exe：跟踪MongDB实例读写时间工具

Hexo常用插件列表

Git部署

Install hexo-deployer-git.

$ npm install hexo-deployer-git --save

Edit settings.

deploy:
  type: git
  repo: <repository url>

RSS分享

安装RSS插件

npm install hexo-generator-feed --save

启用插件
在hexo_config.yml文件添加插件配置项

plugins:
- hexo-generator-feed

升级插件

npm update

卸载插件

npm uninstall hexo-generator-feed

Hexo在github上部署静态Blog

系统环境
Windows 10 专业版

安装前提

Node.js
Git

下载安装

npm install -g hexo-cli

建站

执行下列命令，Hexo创建最简站点

$ hexo init <folder>
$ cd <folder>
$ npm install

文件夹结构

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

_config.yml
网站的配置信息

package.json
npm包

scaffolds
模版 文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。

source
资源文件夹是存放用户资源的地方。除 posts 文件夹之外，开头命名为 (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。

themes
主题 文件夹。Hexo 会根据主题来生成静态页面。

Git for Windows 安装配置

系统环境
Windows 10 专业版

下载安装

https://git-for-windows.github.io/

环境变量

C:\Program Files\Git\cmd;
C:\Program Files\Git\mingw64\bin;C:\Program Files\Git\usr\bin;

配置

申请Github账号
https://github.com/

设置Git的username和email

$ git config --global user.name "username"
$ git config --global user.email "name@org.com"
$ git config --list

生成密钥

$ ssh-keygen -t rsa -C "name@org.com"

Github添加ssh key
Account Settings -> SSH Keys -> Add SSH key

验证ssh key

$ ssh -T git@github.com
