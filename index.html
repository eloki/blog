Hexo常用插件列表

Git部署

Install hexo-deployer-git.

$ npm install hexo-deployer-git --save

Edit settings.

deploy:
  type: git
  repo: <repository url>

RSS分享

安装RSS插件：

npm install hexo-generator-feed --save

启用插件：
在hexo_config.yml文件添加插件配置项

plugins:
- hexo-generator-feed

升级插件：

npm update

卸载插件：

npm uninstall hexo-generator-feed

Hexo在github上部署静态Blog

系统环境
Windows 10 专业版

安装前提

Node.js
Git

下载安装

npm install -g hexo-cli

建站

执行下列命令，Hexo创建最简站点

$ hexo init <folder>
$ cd <folder>
$ npm install

文件夹结构

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

_config.yml
网站的配置信息

package.json
npm包

scaffolds
模版 文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。

source
资源文件夹是存放用户资源的地方。除 posts 文件夹之外，开头命名为 (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。

themes
主题 文件夹。Hexo 会根据主题来生成静态页面。

Git for Windows安装与使用

系统环境
Windows 10 专业版

下载安装

https://git-for-windows.github.io/

环境变量

C:\Program Files\Git\cmd;
C:\Program Files\Git\mingw64\bin;C:\Program Files\Git\usr\bin;

申请Github账号

https://github.com/

设置Git的username和email

$ git config --global user.name "username"
$ git config --global user.email "name@org.com"
$ git config --list

生成密钥

$ ssh-keygen -t rsa -C "name@org.com"

Github添加ssh key

Account Settings -> SSH Keys -> Add SSH key

验证ssh key

$ ssh -T git@github.com
