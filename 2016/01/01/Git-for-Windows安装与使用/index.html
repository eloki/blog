Git for Windows安装与使用

系统环境
Windows 10 专业版

下载安装

https://git-for-windows.github.io/

环境变量

C:\Program Files\Git\cmd;
C:\Program Files\Git\mingw64\bin;C:\Program Files\Git\usr\bin;

配置

申请Github账号
https://github.com/

设置Git的username和email

$ git config --global user.name "username"
$ git config --global user.email "name@org.com"
$ git config --list

生成密钥

$ ssh-keygen -t rsa -C "name@org.com"

Github添加ssh key
Account Settings -> SSH Keys -> Add SSH key

验证ssh key

$ ssh -T git@github.com
