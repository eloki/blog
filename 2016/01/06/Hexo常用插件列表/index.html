Hexo常用插件列表

Git部署

Install hexo-deployer-git.

$ npm install hexo-deployer-git --save

Edit settings.

deploy:
  type: git
  repo: <repository url>

RSS分享

安装RSS插件：

npm install hexo-generator-feed --save

启用插件：
在hexo_config.yml文件添加插件配置项

plugins:
- hexo-generator-feed

升级插件：

npm update

卸载插件：

npm uninstall hexo-generator-feed
