Hexo在github上部署静态Blog

系统环境
Windows 10 专业版

安装前提

Node.js
Git

下载安装

npm install -g hexo-cli

建站

执行下列命令，Hexo创建最简站点

$ hexo init <folder>
$ cd <folder>
$ npm install

文件夹结构

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

_config.yml
网站的配置信息

package.json
npm包

scaffolds
模版 文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。

source
资源文件夹是存放用户资源的地方。除 posts 文件夹之外，开头命名为 (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。

themes
主题 文件夹。Hexo 会根据主题来生成静态页面。
