MongoDB for Windows安装配置

系统环境
Windows 10 专业版

下载安装

https://www.mongodb.org/downloads

环境变量

C:\Program Files\MongoDB\Server\3.2\bin;

配置

创建数据库和日志存放目录
新建“MongoDB\db”和“MongDB\log”两个文件夹，分别存放数据库文件和日志文件

创建config文件
“C:\Program Files\MongoDB 2.6 Standard\bin”,新建mongo.config文件,内容:

##数据库
dbpath=D:\ProgramData\MongoDB\db
##日志
logpath=D:\ProgramData\MongoDB\log\mongodb.log

添加环境变量

PATH="C:\Program Files\MongoDB\Server\3.2\bin;"

安装服务
以管理员方式打开CMD窗口，安装MongoDB服务

mongod --config "C:\Program Files\MongoDB\Server\3.2\bin\mongo.config" --install

启动服务

net start mongodb

测试连接

mongo

文件结构

C:\Program Files\MongoDB\Server\3.2\bin\
mongo.exe：客户端，支持js语法
mongod.exe：服务端
mongodump.exe：备份工具
mongorestore.exe：恢复工具
mongoexport.exe：导出工具
mongoimport.exe：导入工具
mongostat.exe：实时性能监控工具
mongotop.exe：跟踪MongDB实例读写时间工具
